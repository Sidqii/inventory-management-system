# Inventory Management System

A refactored and simplified update of an Inventory Management System created during my previous internship.

## Feature Project

- Authentication
- Categories
- Products
- Warehouses
- Suppliers
- Stock Management
- Stock Requests
- Stock Movements
- Role-Based Access Control (RBAC)
